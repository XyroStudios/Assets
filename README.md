
Xyro Studios - Internal Recruitment API v4.2
This repository contains the source code for the Xyro Identity Verification Portal. This is an internal tool used by the HR department to verify the digital signatures of incoming contractors and developers.

📋 Features
AES-256 Session Encryption

OAuth2.0 Integration

Real-time hardware verification

⚙️ Installation
This is a private deployment. For instructions on how to use the signature portal, please refer to the Internal Onboarding Document (Google Docs) sent by your recruiter.
